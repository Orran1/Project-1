Notes

The delete a line = hold command + x or control + x
To generate words = write lorem follow by number of words you want generate, example Lorem30 to generate 30 words.
To open link in a new tab = write target="_blank" in anchor element to open link in a new tab
To open inspec element = crtl+shift+c and to close crtl+shift+i
To move a line = alt or option + up or down
To copy a line = alt or option + shift + down or up
To delete a line = hold ctrl or command + x or control + x
Extensions for VScode = Live Server, Mithril Emmet, Prettier
How to print an images = for local images ./ and for online images ../ (move the image to workspace first)
To look for folders in the same directory =./ , to look for outside of directory ../
Input textbox = textarea element followed by rows="10" for 10 rows
Horizontal line = hr tag
90% of all a time, you'll use these elements = nav,section,h1,h2,form,input,textarea,button,main,footer,img,div,a,p
